# MCP Strategy

## Ordem obrigatória de recuperação

Antes de ler arquivos:

1. Procurar símbolo
2. Procurar função
3. Procurar classe
4. Ler trecho específico
5. Expandir contexto apenas se necessário

## Proibido

- Ler src inteiro
- Ler docs inteiro
- Ler repositório inteiro

Sem justificativa explícita
